# Quad Squad

A `quad` draws a four-sided polygon. It needs 8 parameters — four (x, y) vertices.

---

## Task 1 — Basic Quad

Draw a trapezoid using `quad(x1, y1, x2, y2, x3, y3, x4, y4)`.

Vertices: (150, 100), (250, 100), (300, 300), (100, 300).

**Validation:** `quad(8)` + `pixelMatch(200, 200, [180, 130, 255])`
