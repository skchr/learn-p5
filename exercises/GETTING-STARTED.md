## Setting up the canvas

Before we can draw anything onto the screen, we need to setup our canvas.

The setup function allows us to set the dimension of our canvas and run any other code which is important to call before we begin to draw anything. For example, we can setup the background of our canvas:

For simplicity, all the starter code will call `createCanvas` and declare the canvas size for the user.  

```js

function setup(){

background("pink")


}



```

### Instructions

Change the color to any color string.
